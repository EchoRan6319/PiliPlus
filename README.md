### 关于字体问题修复

本仓库的主要目的之一是解决PiliPlus从Flutter 3.38版本起在ColorOS设备上出现的字体调用异常问题（系统字体被错误替换为ColorOS衬线字体OPPO Serif）。

**使用说明**：
- 安装应用后，请进入设置界面的"外观设置"中，开启"使用OPPO Sans字体"开关
- 开启后，应用内所有文本元素（包括弹幕）都会使用OPPO Sans字体，解决字体显示异常问题

- **最新版ColorOS16.0.5.xxx已经修复了此问题，本仓库永久停更**
